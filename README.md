-- Οδηγείες χρήσεις --

Τοποθετήστε τον φάκελο της εργασίας "prototype" στον φάκελο www (για WAMP) ή htdocs (για XAMPP)
Ξεκινήστε το Apache και το MySQL από το control panel του WAMP ή του XAMPP

-- Οδηγείες δημιουργίας Βάσης Δεδομένων --

Υπάρχουν δυο επιλογές για την δημιουργία της Βασής Δεδομένων.
Παρακάτω παρουσιάζονται οι οδηγίες για καθεμία.

-- 1ή Επιλογή. Αυτόματη Δημιουργία μέσω create_db.php

Ανοίξτε τον browser και εισάγεται την παρακάτω διεύθυνση στο address bar:
http://localhost/prototype/Ergasia/create_db.php

Αν η δημιουργία της βάσης δεδομένων ολοκληρώθηκε επιτυχώς μάζι με τα μηνύματα επιτυχείας θα εμφανιστεί και ένας
υπερσύνδεσμος που οδηγεί στην αρχική σελίδα της εφαρμογής.

-- 2ή Επιλογή. Χειροκίνητη Εισαγωγή μέσω SQL

Ανοίξτε τον browser και εισάγεται την παρακάτω διεύθυνση στο address bar:
http://localhost/phpmyadmin

Δημιουργήστε νέα βάση δεδομένων με όνομα schoolDB

Επιλέξτε Import και εισάγετε το αρχείο schoolDB.sql από τον φάκελο prototype/Ergasia


-- Εκκίνηση Εφαρμογής

Η βάση δεδομένων μπορεί να προβληθεί εδώ: 
"http://localhost/phpmyadmin"


Για να μεταβείτε στην αρχική σελίδα της εφαρμογής εισάγετε την παρακάτω διεύθυνση στο browser:
http://localhost/prototype/Ergasia/index.php

Τώρα μπορείτε να αρχίστε το πρόγραμμα κανονικά.
